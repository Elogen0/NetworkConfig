## 쿠버네티스 환경을 위하여 아래 설정을 해줍니다.
```
# swap이 켜져있으면 메모리가 스토리지 자원을 끌어 쓸수 있기에 쿠버네티스에서 정확한 메모리 리소스를 유추할 수 없습니다.
# 쿠버네티스의 안정적인 동작을 위해 스왑(Swap) 메모리를 비활성화합니다.
sudo swapoff -a

# 재부팅 시에도 자동 마운트가 해제 되게 하여 스왑이 활성화되지 않도록 설정 파일에서 주석 처리합니다.
# Regex로 문자열를 치환하는 것으로, /etc/fstab파일을 열어 
# /swap.img none  swap sw 0  0 줄앞에 주석(#)을 걸어 비활성화 합니다.
sudo sed -i '/ swap / s/^\(.*\)$/#\1/g' /etc/fstab

# 컨테이너 네트워크 통신을 위한 커널 모듈(overlay, br_netfilter)을 로드 설정에 추가합니다.
cat <<EOF | sudo tee /etc/modules-load.d/k8s.conf
overlay
br_netfilter
EOF

# 설정한 커널 모듈을 시스템에 즉시 로드합니다.
sudo modprobe overlay
sudo modprobe br_netfilter

# 커널모듈 및 브릿지를 활성화 하여 네트워크를 통한 트래픽이 iptables에 의해 제어되도록 커널 파라미터를 설정합니다.
cat <<EOF | sudo tee /etc/sysctl.d/k8s.conf
net.bridge.bridge-nf-call-iptables  = 1
net.bridge.bridge-nf-call-ip6tables = 1
net.ipv4.ip_forward                 = 1
EOF

# 변경된 커널 파라미터 설정을 시스템에 즉시 적용합니다.
sudo sysctl --system
```

## 추가) sudo 명령어 안쳐도 되는 설정
`sudo visudo` 쳐서 나오는 파일 맨 마지막 줄에 추가 ($USER_NAME은 현재 유저이름으로 바꿔서)
```
$USER_NAME ALL=(ALL) NOPASSWD: ALL
```
